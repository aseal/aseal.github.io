Guessing game

For many people, guessing game is their first game program. This is a 'Hello World' of game programming. Here I added some twists to the basic guessing game, but nothing fancy.

Guess the number

The objective of the game is to guess an 8 digit number. Everytime you pick a number you will get a hint. The hint is a number which tells how many digits of the guessed number are at the same place as of the correct number.

The score keeps track of number of guesses.

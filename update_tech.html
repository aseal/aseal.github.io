Flycut

Aug 03, 2016

There are several occasions where I need to copy multiple texts from one appliacation to another application. So far, I had to go back and forth between applications and copy paste one item at a time. If something goes wrong repeat the whole process again. Very frustrating. Then I found out about Flycut from of my colleague. Voilà!! You can download Flycut for MAC from https://www.macupdate.com/app/mac/42134/flycut

With Flycut, you copy text just like you normally would (Command-C). Everytime you copy or cut text it’s stored in FlyCut’s stack. To paste an item from FlyCut, press Shift-Command-V (instead of Command-V), then you can select the item to paste from a list by using arrow keys.

It is simple and sweet with no unnecessry feature overload. However, I wish there was a way to control the copy also. I don't want to copy everything to flycut and leave it there (for security reasons).

OBi100 firmware update

Jul 24, 2016

My Obihai connected phone suddenly stopped working. After googling figured out the current firmware is no longer compatible with google voice. The steps to update the firmware were pretty stratight-forward and worked as expected.

  • Download latest firmware from http://fw.obihai.com/OBi-latest.fw.
  • Dial ***1 from the phone attached to OBi and note the IP address that it reads.
  • Open a web browser to that address.
  • The default username/password (if not updated), is admin/admin.
  • Go to System Management -> Device Update -> Firmware Update -> Select file to update firmware.
  • Click "Update" to start the update.
Once the update was finished, the phone was back to life.

Back to My MAC

Jan 05, 2016

For several years, I used port forwarding to connect to my home network. Leaving a port open is not good idea. It was hassle to open and close before and after use. Now I've multiple MACs. So I've switched to Back to My MAC feature

It can be set up by selecting System Preferences -> iCloud -> Back to My Mac. The remote Mac will appear under the Shared section of the sidebar in Finder.

When I was exploring this, I found there are few other ways as well -

  • Mac contains a Screen Sharing feature, which is essentially a VNC server with some extra features. This allows standard VNC clients to control your Mac. It can be enabled at System Preferences -> Sharing -> Screen Sharing
  • Chrome Remote Desktop extentsion
  • Using TeamViewer (just like logmein which is a paid service now.)

How to model an optimization problem

Dec 18, 2015

The first step towards solving an optimization problem, is modeling the problem. Here are the basic steps(with example of Knapsack problem).

  • Choose some decision variables
    • The decision variables (xi) select if the item to be included.
  • Express the problem constraints in terms of these variables
    • In case of knapsack problem, the constraint is max capacity
  • Express the objective function
    • Maximize the value in knapsack

Diameter of a tree

Aug 22, 2015

How to find the longest path (diameter) in a tree? For, binary tree, this can be achieved by recursively vising left and right nodes and calculating their diameters (link). For, n-ary tree, we can use brute-force and run bfs/dfs on all nodes recording the max distance. But there is a very efficient way.

  • Run BFS/DFS on any node to find the farthest leaf node.
  • Run BFS/DFS again on the previously found farthest node.
  • The longest path from the second run is the diameter of a tree

The main idea here is that, the farthest point found in BFS/DFS starting from any node is actually one of the nodes in one of the longest paths. If you are looking for proof, you can find it here.
